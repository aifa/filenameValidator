eSubmission Tools

(beta)

Simple tools for creating and validating electronic submission file names for centralized procedures and periodic safety assessment reports. These can be used by anyone interested in submitting files to the European Medicines Agency eSubmissions systems. The rules and conventions applied for the creation and validation of the filenames can be found in this guide.
